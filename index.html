Ai-Levator

Elevator Experience Design

Overview
Visual Approach
Deliverables
Team

Project Vision

Ai-Levator reimagines the elevator experience through intuitive design and visual communication. The project aims to create a system that not only reduces wait times but transforms how people interact with vertical transportation in buildings.

The core of this project is creating visual interfaces that make complex systems feel simple and intuitive for users. By combining smart technology with thoughtful design, we're aiming to create elevator experiences that are efficient, accessible, and even enjoyable.

Key Beneficiaries

  • Building Occupants: Employees and residents who use elevators daily
  • Visitors: People unfamiliar with the building who need clear guidance
  • People with Accessibility Needs: Those who require inclusive design considerations

Visual Design Opportunities

  • Creating an intuitive interface system for elevator controls
  • Designing visual guidance cues for elevator lobbies
  • Developing motion graphics for interactive displays
  • Creating storyboards for user flow sequences
  • Designing a visual language that works across digital and physical touchpoints

Visual Communication Strategy

The Ai-Levator project focuses on creating visual systems that make complex elevator routing accessible to all users. Our approach emphasizes:

  • Visual Hierarchy: Using size, color, and placement to prioritize information
  • Motion Design: Creating animations that indicate system status and direction
  • Consistent Visual Language: Developing a cohesive design system across all touchpoints
  • Inclusive Design: Ensuring the system works for users with varying abilities

Design Focus Areas

  • Control Panel Interface: Reimagining how users select floors and understand system status
  • Lobby Displays: Creating informative visuals that guide users to the right elevator
  • Mobile Integration: Designing a companion app interface that connects with the building system
  • Status Indicators: Developing visual cues that show elevator location and status
  • Experience Storyboarding: Planning the complete user journey through visual sequences

Creative Challenges

  • How might we use visual design to make waiting times feel shorter?
  • What visual cues can efficiently guide users to their assigned elevator?
  • How can we create intuitive interfaces that require minimal learning?
  • What animation principles could help users understand elevator movement?
  • How might we storyboard the ideal elevator experience from entry to exit?

Expected Outputs

The Ai-Levator project will create several key deliverables focused on the visual and experiential aspects of elevator systems:

  • Interface Designs: Visual mockups for control panels and digital displays
  • Experience Storyboards: Visual sequences showing user journeys through the system
  • Motion Prototypes: Animated demonstrations of key interactions
  • Visual System Guidelines: Design specifications for colors, typography, and iconography
  • Mobile App Interface: Design for a companion application

Visual Components

  • Iconography Set: Custom icons to represent system functions and status
  • Color System: Accessible color palette for interfaces and signage
  • Animation Library: Motion patterns for different system states
  • Wayfinding Elements: Visual cues to guide users through the space
  • Digital and Physical Integration: Design specifications for how digital interfaces connect with physical elements

Design Opportunities

Creative team members will have the chance to develop:

  • Comprehensive interface mockups for multiple touchpoints
  • Animated prototypes demonstrating key interactions
  • Storyboards illustrating the complete user journey
  • Visual identity elements that create a cohesive system
  • Presentation materials that effectively communicate design concepts

Project Vision

Fallacy Hunter transforms how people identify logical fallacies by making abstract concepts visually concrete. This browser extension helps users recognize flawed reasoning through intuitive visual cues and illustrated examples.

Our goal is to create a tool that makes critical thinking skills more accessible through strong visual communication, helping users identify reasoning errors and strengthen their own arguments.

Key Beneficiaries

  • Students: Learning critical thinking and argumentation
  • Educators: Teaching logical reasoning in engaging ways

  • General Public: Evaluating information more critically
